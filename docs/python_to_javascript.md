## Classes

- **constructor**: Same as `__init__`
  Python:

  ```py
  class User():
      def __init__(self, name):
          self.name = name
  ```

  JavaScript:

  ```js
  class User {
    constructor(name) {
      this.name = name;
    }
  }
  ```
